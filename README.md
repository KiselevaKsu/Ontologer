# Ontologer
Установка
1. Скачайте PDF учебника: https://drive.google.com/file/d/1ctTzxcZkAJsvSaX0jVDXv2cazCJ7ggIX/view?usp=sharing
2. Скачайте модель LLM: https://drive.google.com/file/d/1H8mS1a6yn57Nd7tKU1GmZtP_9JerImXQ/view?usp=sharing
3. Поместите оба файла в корень проекта
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите сервер: `python server.py`
