# Ontologer

Веб-приложение с двумя режимами:
- Консультация — поиск ответов по учебнику
- Тестирование — проверка знаний с автоматической оценкой

Установка
1. Скачайте PDF учебника: https://drive.google.com/file/d/1kPAtjW-JfaqJL0X9_kkYlpzOWTuaSm_z/view?usp=share_link
2. Поместите файл в корень проекта
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите сервер: `python server.py`
